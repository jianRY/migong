## 🎯 游戏玩法

### 目标
找到从起点到终点的正确路径，抵达紫色光点即可获胜！

### 操作方式
| 输入方式 | 操作 |
|---------|------|
| 键盘 | ↑↓←→ 或 WASD 移动 |
| 触屏 | 滑动控制 |
| R 键 | 重新开始 |
| Esc | 暂停/继续 |
| 空格 | 开始/继续 |

## ✨ 功能特色

### 🎲 迷宫系统
- **随机生成** - DFS 算法，每次都是全新的迷宫
- **动态难度** - 5×5 到 41×41 多级选择
- **唯一解** - 保证有解路径

## 🛠️ 技术实现

### 核心技术
- **Canvas 2D** - 游戏渲染
- **DFS 算法** - 迷宫生成
- **Web Audio API** - 音效合成
- **requestAnimationFrame** - 动画循环

### 代码结构
```
maze.html (单一文件)
├── CSS 样式 (霓虹/赛博朋克风格)
├── 游戏逻辑 (迷宫生成、移动、碰撞检测)
├── 渲染系统 (墙壁/玩家/终点绘制)
├── 20种样式切换
└── 粒子/烟花特效
```

### 性能优化
- 单一 HTML 文件，无需构建
- Canvas 渲染，高效流畅
- 移动端适配
