# 🎮 霓虹迷宫

一个精美的 HTML5 迷宫游戏，采用赛博朋克风格设计。

## 🎯 游戏玩法

- 使用 **方向键** 或 **WASD** 控制移动
- 到达 **紫色光点** 即可获胜
- 触屏设备支持滑动控制

## ✨ 功能特色

- 🎲 **随机迷宫** - 每次生成不同的迷宫路线
- 📏 **多级难度** - 从 5×5 到 41×41 可选
- 🎨 **20种墙壁样式** - 霓虹墙、像素墙、卡通墙、火焰墙...
- 🐱 **20种玩家图标** - 兔子、老虎、熊猫、狐狸...
- 🚪 **20种终点图标** - 门、奖杯、皇冠、钻石...
- 🔊 **音效系统** - 移动、胜利、撞墙音效
- 🎆 **炫酷特效** - 粒子效果、烟花庆祝

## 🕹️ 操作说明

| 按键 | 功能 |
|------|------|
| ↑↓←→ / WASD | 移动 |
| R | 重新开始 |
| Esc | 暂停/继续 |
| Space | 开始/继续 |
