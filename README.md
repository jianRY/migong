## 🎯 游戏目标

找到从起点到终点的正确路径，抵达**紫色光点**即可获胜！

## 🕹️ 操作说明

| 输入 | 操作 |
|------|------|
| ↑↓←→ / WASD | 移动 |
| 触屏滑动 | 移动 |
| R | 重新开始 |
| Esc | 暂停/继续 |
| 空格 | 开始/继续 |

## ✨ 功能特色

### 🎲 迷宫系统
- **随机生成** - DFS 算法
- **动态难度** - 5×5 到 41×41
- **唯一解** - 保证有解路径

## 🚀 在线预览

直接在浏览器打开: [maze.html](./maze.html)

```bash
# 克隆仓库
git clone https://github.com/jianRY/migong.git

# 打开
cd migong && open maze.html
```
